points.h

Data Types

  • Point:

    A point is represented as a pair of type double

  • Point Set:

    A point set is represented is a vector of points.


Methods

  •  void printPoint( point )
    Prints the X and Y coordinate of a point. 
  •  double distBetween( point, point )
    Calculates the Euclidean distance between given points. 
  •  double angleBetween( point, point, point )
    Determines the smaller angle between two line segments using cosine rule. 
  •  point getInteriorPoint( pointset )
    Returns a point to inside the given pointset. 
  •  point getExteriorPoint( pointset )
    Returns a point outside the given pointset. 
  •  int signedArea( point, point, point )
    Calculates the signed are contained by the given three points in that order. 
  •  bool checkRightTurn( point, point, point )
    Determines the direction of rotation of given three points in that order.
