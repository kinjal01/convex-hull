Convex Hull Algorithms

Objective:

In this assignment we implement three different Convex Hull algorithms, namely:

  1. Graham Scan Algorithm
  2. Jarvis March Algorithm
  3. Andrew Algorithm

We intend to compare the performance and accuracy of these algorithms for various test cases and also analyse behaviour of the algorithms in degenerate cases. We then present our rationale for determining the algorithm that outperforms the other two.

Design Principles

Here we take a look at a breakdown of our code and explain our approach.

  • Driver Program

    Driver program takes as input from STDIN:
    Number of points, N.
    X and Y coordinate of N points.
    The points are stored as a pointset. ConvexHull API call returns a subset of these points present in the ConvexHull in cyclic order (unique to each algorithm).

  • algorithm.h

    algorithm.h contains the implementation of the aforementioned three algorithms. The corresponding methods can be invoke by calling:

    pointset convexHullGrahamScan( pointset )
    pointset convexHullJarvisMarch( pointset )
     
    pointset convexHullAndrew( pointset )
     All three methods take one argument, pointset as read by the Driver Program and return a subset of those points.

  • points.h

    points.h provides an abstract representation of a point and a set of points. It also contains methods for providing operations that are required by Convex Hull algorithms.
